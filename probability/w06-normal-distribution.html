  2. When raw scores are turned into $z$-scores, it is known as standardising. The standardised normal distribution will have a mean of 0 and a standard deviation of 1, and can therefore be written:

    $$Z \sim N(0,1)$$

    You can convert normal distributions into standard normal distributions by standardising. This allows you to calculate proportions of the population that are more than, less than, or between values. You can calculate these probabilities using tables, software, or websites such as at this link or this one.

      3. $P(Z > 0.45)$
      4. $P(-1.2 < Z < 1.6)$
      Show Solution

      i. $P(Z < 1)$:
      Using a calculator for the standard normal distribution, we find the area to the left of $z=1$.

      Answer: 0.8413

      ii. $P(Z < -0.45)$:
      Using a calculator, we find the area to the left of $z=-0.45$.

      Answer: 0.3264

      iii. $P(Z > 0.45)$:
      Using a calculator for a right-tail probability, we find the area to the right of $z=0.45$.

      Answer: 0.3264

      iv. $P(-1.2 < Z < 1.6)$:
      This is calculated as $P(Z < 1.6) - P(Z < -1.2)$.
      Using a calculator, $P(Z < 1.6) = 0.9452$ and $P(Z < -1.2) = 0.1151$.
      $0.9452 - 0.1151$.

      Answer: 0.8301

  3. Given the information about the population distribution, find the probabilities requested by first standardising the raw scores.

    1. $X \sim N(80,25)$

      1. $P(X < 85)$
      2. $P(X > 70)$
      3. $P(78 < X < 82)$
      Show Solution

      The standard deviation is $\sigma = \sqrt{25}=5$.

      i. $P(X < 85)$:
      Standardise: $Z = \frac{85 - 80}{5} = 1$.
      Find $P(Z < 1)$ using a calculator.

      Answer: $0.8413$

      ii. $P(X > 70)$:
      Standardise: $Z = \frac{70 - 80}{5} = -2$.
      Find the right-tail probability $P(Z > -2)$ using a calculator.

      Answer: $0.9772$

      iii. $P(78 < X < 82)$:
      Standardise both: $Z_1 = \frac{78 - 80}{5} = -0.4$. $Z_2 = \frac{82 - 80}{5} = 0.4$.
      Find the 'between' probability $P(-0.4 < Z < 0.4)$ using a calculator.

      Answer: $0.3108$

    2. $X \sim N(150, 156.25)$

      1. $P(X < 143.75)$
      2. $P(X > 180)$
      3. $P(145 < X < 170)$
      Show Solution

      The standard deviation is $\sigma = \sqrt{156.25}=12.5$.

      i. $P(X < 143.75)$:
      Standardise: $Z = \frac{143.75 - 150}{12.5} = -0.5$.
      Find $P(Z < -0.5)$ using a calculator.

      Answer: $0.3085$

      ii. $P(X > 180)$:
      Standardise: $Z = \frac{180 - 150}{12.5} = 2.4$.
      Find the right-tail probability $P(Z > 2.4)$ using a calculator.

      Answer: $0.0082$

      iii. $P(145 < X < 170)$:
      Standardise both: $Z_1 = \frac{145 - 150}{12.5} = -0.4$. $Z_2 = \frac{170 - 150}{12.5} = 1.6$.
      Find the 'between' probability $P(-0.4 < Z < 1.6)$ using a calculator.

      Answer: $0.6006$

  4. Given the information about the population distribution, find the values that are represented by the following probabilities/proportions. Give your answers correct to 1 decimal place. You will need to find the $z$-score from the probability given and then transform this back to a raw score.

    1. $X \sim N(100, 16)$

      1. $P(X < r) = 0.9772$
      2. $P(X > s) = 0.9772$
      3. $P(X < t) = 0.35$
      Show Solution

      The standard deviation is $\sigma = \sqrt{16}=4$.

      i. Find r:
      Use an inverse normal calculator for a left-tail probability of $0.9772$ to find $z \approx 2.0$.
      Rearrange the formula to find the raw score: $r = \mu + z\sigma = 100 + (2.0)(4)$.

      Answer: $r = 108.0$

      ii. Find s:
      Use an inverse normal calculator for a right-tail probability of $0.9772$ to find $z \approx -2.0$.
      Rearrange: $s = \mu + z\sigma = 100 + (-2.0)(4)$.

      Answer: $s = 92.0$

      iii. Find t:
      Use an inverse normal calculator for a left-tail probability of $0.35$ to find $z \approx -0.39$.
      Rearrange: $t = \mu + z\sigma = 100 + (-0.39)(4)$.

      Answer: $t \approx 98.4$

    2. $X \sim N(20, 4)$

      1. $P(X < r) = 0.9772$
      2. $P(X > t) = 65\%$
      Show Solution

      The standard deviation is $\sigma = \sqrt{4}=2$.

      i. Find r:
      Use an inverse normal calculator for a left-tail probability of $0.9772$ to find $z \approx 2.0$.
      Rearrange: $r = \mu + z\sigma = 20 + (2.0)(2)$.

      Answer: $r = 24.0$

      ii. Find t:
      We are given the right-tail probability $P(X > t) = 0.65$.
      Use an inverse normal calculator for a right-tail probability of $0.65$ to find $z \approx -0.39$.
      Rearrange: $t = \mu + z\sigma = 20 + (-0.39)(2)$.

      Answer: $t \approx 19.2$

  5. A hot drinks machine is known to dispense liquid in a volume that is normally distributed with a mean of 250 ml and a standard deviation of 5 ml, so $V \sim N(250, 25)$.

    1. What proportion of drinks are expected to be dispensed that are over 254 ml?

      Show Solution

      We need $P(V > 254)$.
      Standardise: $Z = \frac{254 - 250}{5} = 0.8$.
      Find the right-tail probability $P(Z > 0.8)$ using a calculator.

      Answer: $0.2119$ or $21.19\%$

    2. What proportion of drinks are expected to be under 246 ml in volume?

      Show Solution

      We need $P(V < 246)$.
      Standardise: $Z = \frac{246 - 250}{5} = -0.8$.
      Find the left-tail probability $P(Z < -0.8)$ using a calculator.

      Answer: $0.2119$ or $21.19\%$

    3. Use your answers and sketches to parts a and b to work out the probability that a drink is dispensed that is between 246 ml and 254 ml.

      Show Solution

      We need $P(246 < V < 254)$, which corresponds to $P(-0.8 < Z < 0.8)$.
      This is the total probability (1) minus the two tail areas we found in parts a and b.
      $1 - P(V > 254) - P(V < 246) = 1 - 0.2119 - 0.2119$.

      Answer: $0.5762$ or $57.62\%$

    4. What volume are 75% of drinks less than?

      Show Solution

      We need to find $v$ such that $P(V < v) = 0.75$.
      Use an inverse normal calculator for a left-tail probability of $0.75$ to find $z \approx 0.674$.
      Rearrange: $v = \mu + z\sigma = 250 + (0.674)(5)$.

      Answer: $253.4$ ml (approx)

    5. You find that your cup overflows 2% of the time. How big is your cup?

      Show Solution

      Overflowing means the volume is greater than your cup size, $c$.
      We have the right-tail probability $P(V > c) = 0.02$.
      Use an inverse normal calculator for a right-tail probability of $0.02$ to find $z \approx 2.054$.
      Rearrange: $c = \mu + z\sigma = 250 + (2.054)(5)$.

      Answer: $260.3$ ml (approx)
